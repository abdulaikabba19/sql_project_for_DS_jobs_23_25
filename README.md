




## **1. Introduction and Overview**
This project contains a collection of SQL queries designed to analyze Data Science job postings from **2023 to 2025**. There has been a project of this same nature that I worked on, but was only limited to 2023.  
Each query serves a specific analytical purpose — from cleaning and transforming job‑market data to extracting insights about skills, salaries, job types, and hiring trends. Please find the attached folder here: [updated_sql_folder](/updated_project_sql/)


This project demonstrates how SQL can be used as a powerful tool for **data exploration**, **feature extraction**, and **evidence‑based career insights** for aspiring and practicing Data Scientists. The data, support, and guidance  for this project was through [SQL Course](https://lukebarousse.cpm/sql) 


---

## **2. Background**
Between 2023 and 2025, the Data Science job market experienced rapid shifts driven by:

- The rise of **generative AI** and automation  
- Increased demand for **cloud‑native** and **production‑ready** ML skills  
- A growing emphasis on **data engineering** capabilities  
- Hybrid and remote work restructuring  
- Salary adjustments due to economic fluctuations  

To understand these changes, job‑posting datasets were analyzed using SQL.  
The queries in this folder were written to answer questions such as:

- Which skills are most in demand?  
- How do salaries vary by role, skill, and location?  
- What trends emerged between 2023 and 2025?  
- Which tools and technologies consistently appear in high‑paying roles?

## **4. The Analysis**
Each SQL file in this folder performs a targeted analytical task. Examples include:

- **Skill Demand Queries**  
  Identify the most frequently requested skills (Python, SQL, R, AWS, Tableau, etc.) across 2023–2025.

- **Salary Insights**  
  Compare average salaries by skill, job title, and work‑from‑home status.

- **Trend Analysis**  
  Track how demand for specific skills changed over the three‑year period.

- **Role‑Specific Breakdowns**  
  Extract insights for Data Scientist, ML Engineer, Data Analyst, and AI‑focused roles.

- **Data Cleaning & Standardization**  
  Normalize job titles, remove duplicates, and standardize skill names.

- **CTE vs. Aggregation Comparisons**  
  Validate query accuracy by comparing multiple approaches to the same problem.

Together, these queries form a complete SQL‑driven exploration of the Data Science job landscape.

## **6. Conclusions**
The SQL analysis in this folder provides a clear, data‑driven view of the Data Science job market from **2023 to 2025**.  
Key takeaways include:

- **Python + SQL** form the backbone of nearly all DS roles.  
- **Cloud and ML engineering skills** grew sharply in demand.  
- **High‑paying roles** consistently require a blend of analytics, engineering, and production‑ready ML skills.  
- SQL remains one of the most valuable tools for exploring and understanding job‑market datasets.
